# Yashwanth KS - Backend Developer Portfolio (Next.js)

This is a fully working portfolio site built with:

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion animations

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Deploy

You can deploy easily to Vercel:

```bash
npm i -g vercel
vercel
```
