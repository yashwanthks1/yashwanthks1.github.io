Automated RPL/DPL Data Ingestion & Consolidation System

Java · Spring Boot · Spring WebFlux · PostgreSQL · Scheduler · Swagger

Production-style backend system that automates downloading, parsing, normalizing, and consolidating global Restricted Party / Denied Party List (RPL/DPL) datasets across multiple formats (CSV/JSON/XML/Excel).

Includes a reactive ingestion pipeline, incremental update logic (insert/update/delete), a source registry with scheduler, and optimized PostgreSQL persistence with a unified schema.

  • Multi-format ingestion and data normalization.
  • Incremental change detection engine.
  • Scheduler-based automation for each source.
  • High-performance REST APIs with Swagger documentation.
